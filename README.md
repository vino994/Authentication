# User Authentication API (Node.js + Express + Mongoose + JWT)

A simple and clean **User Authentication & Authorization API** built using  
**Node.js, Express.js, MongoDB (Mongoose), and JWT**.  
The project follows the **MVC architecture** and includes **Postman API documentation**.
